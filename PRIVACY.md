# Privacy Policy

**ClearTube** is a browser extension that modifies the YouTube homepage layout.

## Data Collection

We do not collect, store, or transmit any user data. This includes:

- No browsing history
- No personal information
- No usage statistics
- No cookies

## Network Traffic

ClearTube generates zero outgoing network requests. It operates entirely within your browser and never communicates with any external server.

## Changes

This policy may be updated if the extension's functionality changes. Any updates will be reflected in this document.

## Contact

If you have any questions, open an issue on the GitHub repository.
